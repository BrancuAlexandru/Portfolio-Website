This is my portfolio website. I designed and coded it myself.

### Technologies
 - React
 - Redux
 - TypeScript
 - SCSS
 - Babel.js
 - Webpack
 - Firebase (Hosting)

### Design
 - Adobe Photoshop CC

### Installation
 - Terminal: npm install
 - Terminal: firebase init
 - Choose /dist/ as your Firebase project's public folder.
 - Terminal: npm run build
 - Terminal: firebase deploy
